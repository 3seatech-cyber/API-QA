Qué contiene esta página de GitHub

Backend API

Servicio FastAPI con documentación Swagger automática, modelos Pydantic y endpoints listos para pruebas académicas y extensión industrial.

Uso sugerido

El backend se ejecuta en FastAPI y la documentación se sirve desde /docs. La página estática en la carpeta docs/ se publica directamente con GitHub Pages para presentar el MVP y consumir la API desplegada.

Despliegue

Cómo publicarlo en GitHub Pages

Página estática

  1. Sube el repositorio a GitHub.
  2. Ve a Settings → Pages.
  3. Selecciona la rama principal y la carpeta /docs.
  4. Guarda y publica.

Backend

  1. Despliega FastAPI en Render, Railway, Fly.io o un VPS.
  2. Habilita CORS para tu dominio de GitHub Pages.
  3. Copia la URL base en el campo de demo de esta página.
